## Retrieve

Retrieves data for a specific step in a workflow

### Example Usage

```csharp
using Novu;
using Novu.Models.Components;

var sdk = new NovuSDK(secretKey: "YOUR_SECRET_KEY_HERE");

var res = await sdk.Workflows.Steps.RetrieveAsync(
    workflowId: "<id>",
    stepId: "<id>"
);

// handle response
```
